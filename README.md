# Condicionales-en-Python
Explicación del Código:

En el archivo `condicionales.py`, se solicita al usuario que ingrese un número entre 1 y 100. Luego, se usan condicionales para determinar si el número ingresado está dentro de un límite permitido, en el rango permitido, o fuera de rango.

Respuestas a las preguntas:

1. **¿Qué tipo de condición te resultó más intuitiva?**
   La condición más intuitiva fue la de verificar si el número ingresado es exactamente 1 o 100, ya que estas son las condiciones de límite. Para mí, comparar un valor exacto es más sencillo de entender.

2. **¿En qué caso usarías condicionales anidadas en un programa real?**
   Usaría condicionales anidadas cuando necesite verificar una condición y, dentro de esa condición, realizar verificaciones adicionales. Un ejemplo real sería en un sistema de validación de acceso donde primero verifico si un usuario está registrado, y luego dentro de esa condición, verifico si la contraseña es correcta.
